Travel Companion

Travel Companion is an acommdation finder and booking system all in one single page application. Users can input their desired options which then filters accommodation to best suits their needs. Upon selection of the accommdation they would like to book a summary is displayed which also includes the total of their stay. Vanilla JavaScript and jQuery were used to create this web app along with following a UX process. This was created during my time at Yoobee School of Design.
